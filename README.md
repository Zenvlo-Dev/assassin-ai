## 🛠️ Getting Started

### Prerequisites
- A modern web browser
- Git installed locally
- A GitHub account

### Installation

```bash
# Clone the repository
git clone https://github.com/Zenvlo-Dev/assassin-ai.git

# Navigate into the project
cd assassin-ai

# Open index.html in your browser
open index.html
# Or on Windows:
# start index.html
```

### Local Development
Since the site uses zero build tools, simply edit the HTML, CSS, or JS files and refresh the browser. No bundlers, transpilers, or package managers needed.
